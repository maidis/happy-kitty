# Introduction #

A video game console emulator is a program that allows a computer or modern console to emulate a different video game console's behavior. Emulators are most often used to play older video games on personal computers and modern video game consoles.


# Details #

In this mission we are going to package all emulators for Pardus. it sounds fun :) Just pick one from list, inform others, start packaging and add package files to Happy Kitty repository.
